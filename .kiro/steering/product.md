# プロダクト概要

Sugu は「すぐ実行できる」をコンセプトにした JavaScript 風のインタプリタ言語。Go で実装されており、REPL・ファイル実行・AWS Lambda 上での実行をサポートする。

## コア機能

- **インタプリタ実行**: Lexer → Parser → Evaluator のパイプラインでソースコードを即時評価
- **REPL**: 対話型実行環境で試行錯誤が可能
- **ファイル実行**: `.sugu` ファイルをコマンドラインから直接実行
- **AWS Lambda 統合**: Sugu コードをサーバーレス関数として実行

## 対象ユースケース

- プログラミング言語処理系の学習・実験
- 簡易スクリプトの実行
- サーバーレス環境での軽量スクリプティング

## バリュープロポジション

- JavaScript に親しみのある構文で学習コストが低い
- `mut`/`const` による明示的な可変性制御
- 動的型付けかつシンプルなオブジェクトシステム
- Go 実装による高いポータビリティ
